<a> is a child of <p> in this case because it is contained within <p>/</p>

<p> opens the paragraph and </p> is the closing tag for the paragraph and defines where the paragraph should end

attributes on <a>:
-href: url: "https://google.com": this specifies what url you want to link to
-target: _blank: this describes how the link opens, in this case it opens in a new browser tab
